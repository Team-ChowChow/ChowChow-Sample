# user 모듈

사용자 프로필, 설정, 검색 기록, 급여 기록을 담당하는 모듈입니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `controller/` | 사용자 프로필 조회/수정, 설정 API 엔드포인트 |
| `dto/` | 사용자 정보 요청/응답 DTO |
| `entity/` | Users, UserSettings, SearchLogs, FeedingRecords 엔티티 |
| `repository/` | 사용자 데이터 접근 레이어 |
| `service/` | 프로필 관리, 앱 설정, 급여 기록 비즈니스 로직 |

## 주요 기능

- 사용자 프로필 (이름, 닉네임, 지역, 프로필 이미지)
- 앱 설정 (알림, 다크모드, 검색 기록 저장)
- 반려동물 급여 기록
- 계정 관리 (비밀번호 변경, 탈퇴)
