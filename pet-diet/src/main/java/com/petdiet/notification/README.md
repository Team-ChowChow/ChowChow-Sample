# notification 모듈

사용자 알림(댓글, 좋아요, 레시피 추천, 시스템 알림)을 담당하는 모듈입니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `controller/` | 알림 조회 및 읽음 처리 API 엔드포인트 |
| `dto/` | 알림 요청/응답 DTO |
| `entity/` | Notifications 엔티티 |
| `repository/` | 알림 데이터 접근 레이어 |
| `service/` | 알림 생성, 읽음 처리, 미읽음 카운트 비즈니스 로직 |

## 알림 유형

| 유형 | 설명 |
|------|------|
| `COMMENT` | 게시글에 새 댓글 |
| `LIKE` | 게시글 좋아요 |
| `RECIPE` | 맞춤 레시피 추천 |
| `SYSTEM` | 시스템 공지 |
