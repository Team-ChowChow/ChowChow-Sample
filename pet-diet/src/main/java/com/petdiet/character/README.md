# character 모듈

반려동물 캐릭터 생성 및 육성 시스템을 담당하는 모듈입니다.
앱 활동(식단 이용, 게시글 작성 등)에 따라 경험치가 쌓이고 캐릭터가 성장합니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `controller/` | 캐릭터 조회/생성/업데이트 API 엔드포인트 |
| `dto/` | 캐릭터 생성 요청 및 응답 DTO |
| `entity/` | PetCharacters, CharacterGrowthLogs, CharacterAssets 엔티티 |
| `repository/` | 캐릭터 데이터 접근 레이어 |
| `service/` | 캐릭터 생성, 경험치 적립, 레벨업 비즈니스 로직 |

## 경험치 획득 기준

| 활동 | 경험치 |
|------|--------|
| 식단 추천 이용 | +10 EXP |
| 식단 기록 | +20 EXP |
| 게시글 작성 | +15 EXP |
| 댓글 작성 | +5 EXP |
