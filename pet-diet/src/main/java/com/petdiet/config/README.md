# config 모듈

Spring Boot 애플리케이션의 설정 클래스들을 관리하는 모듈입니다.

## 역할

| 파일 | 역할 |
|------|------|
| `SecurityConfig.java` | Spring Security 설정 (JWT 필터, 권한 설정) |
| `JwtConfig.java` | JWT 토큰 생성/검증 설정 |
| `OpenAiConfig.java` | OpenAI API 클라이언트 설정 |
| `SupabaseConfig.java` | Supabase 연결 설정 |
| `WebMvcConfig.java` | CORS 및 MVC 설정 |
