# ai/diet 모듈

반려동물 맞춤 식단 추천을 처리하는 모듈입니다.
사용자의 반려동물 정보를 기반으로 OpenAI API를 활용하여 안전한 식단을 생성합니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `controller/` | 식단 추천 요청/응답 API 엔드포인트 |
| `dto/` | 식단 추천 요청(반려동물 프로필) 및 응답(레시피) DTO |
| `service/` | OpenAI 연동, 알레르기/질병 필터링, 급여량 계산 비즈니스 로직 |

## 처리 흐름

1. 반려동물 정보 수집 (종, 나이, 체중, 알레르기, 질병, 식단 목적)
2. 금지 식재료 필터링
3. 사용자 요청 재료 분석
4. OpenAI API 기반 레시피 생성
5. 급여량 계산 및 결과 저장
