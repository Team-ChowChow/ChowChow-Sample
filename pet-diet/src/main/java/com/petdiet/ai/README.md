# ai 모듈

AI 기반 서비스를 담당하는 모듈입니다.
OpenAI API를 활용한 식단 추천 및 이미지 생성 기능을 제공합니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `diet/` | AI 기반 반려동물 식단 추천 서비스 |
| `image/` | AI 기반 반려동물 캐릭터 이미지 생성 서비스 |

## AI 활용 방식

1. **식단 추천**: 반려동물 프로필(종, 나이, 체중, 알레르기, 질병)을 기반으로 맞춤 식단을 LLM으로 생성
2. **이미지 생성**: 반려동물 사진을 입력받아 캐릭터 이미지를 생성형 AI로 변환
