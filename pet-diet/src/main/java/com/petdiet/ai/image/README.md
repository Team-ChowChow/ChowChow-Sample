# ai/image 모듈

반려동물 이미지를 AI로 캐릭터화하고, 레시피 조리 과정 이미지를 생성하는 모듈입니다.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `controller/` | 이미지 생성 요청/응답 API 엔드포인트 |
| `dto/` | 이미지 생성 요청(사진 URL, 반려동물 정보) 및 응답(생성 이미지 URL) DTO |
| `service/` | 생성형 AI 연동, 이미지 처리 및 Supabase Storage 업로드 비즈니스 로직 |

## 기능

1. **캐릭터 생성**: 반려동물 사진 → 캐릭터 이미지 변환
2. **레시피 이미지**: 재료·조리 방식 기반 완성 음식 이미지 생성
